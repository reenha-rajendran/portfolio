Latest Projects

Contact Manager App

In this project, I built a Contact Manager App using React, focusing on key concepts such as component structure, state management, and routing. I started by developing the foundational components that form the app's structure, ensuring proper use of props and state. I then set up navigation between pages using React Router v6, enabling smooth user flow. To manage data, I integrated APIs with Axios, allowing the app to retrieve and display contact information dynamically. I also implemented a search bar, providing users with a way to search through contacts efficiently. As part of the final step, I refactored the application to use the Context API for state management, streamlining data flow across the app. This refactor also ensured that the app adhered to modern React best practices. Throughout, I focused on clean coding, troubleshooting issues, and ensuring all routing and functionality worked seamlessly.

Food Order App

In this project, I developed a Food Order App using React, designed to allow users to browse a variety of food items, add them to a cart, modify quantities, and place orders. I created a separate interface for admins, where they can manage the menu by adding, removing, and viewing food items and orders. I implemented state management using React hooks like useState and useReducer, ensuring dynamic user interactions. The app’s interface was built with Material UI components for a modern, responsive design. I also integrated the Context API to streamline data flow between components and used cloud storage to handle image uploads for food items. Throughout the project, I focused on functionality, user experience, error handling, and clean code organization to meet the project’s specifications and grading criteria.
